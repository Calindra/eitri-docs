# Exposed APIs

As Exposed APIs do Eitri Shopping são métodos complementares aos disponíveis na [Eitri Bifrost](../conceitos/eitri-bifrost.md) para invocar código na camada nativa em aplicativos Eitri Shopping.

Para mais detalhes acesse nossa [Documentação](https://cdn.83io.com.br/library/eitri-shopping-exposed-api-doc/doc/latest/index.html){:target="\_blank"}
